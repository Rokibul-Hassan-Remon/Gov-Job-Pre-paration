```text title:"Communication Protocol"
Communication System

├── 1. Multiplexing Techniques
│       │
│       ├── FDM (Frequency Division Multiplexing)

│
│

│       │
│       ├── Channelization Protocols/Techniques

│       │       └── SC-FDMA
│       │
│       │
│       ├── Random Access Protocols
│       │       │
│       │       ├── ALOHA
│       │       ├── Slotted ALOHA
│       │       ├── CSMA
│       │       ├── CSMA/CD
│       │       └── CSMA/CA
│       │
│       │
│       └── Controlled Access Protocols
│               │
│               ├── Reservation
│               ├── Polling
│               └── Token Passing
│
│
├── 3. Networking / Communication Protocols
│       │
│       ├── Application Layer
│       │       ├── HTTP
│       │       ├── HTTPS
│       │       ├── FTP
│       │       ├── SMTP

│       ├── Transport Layer
│       │       ├── TCP

│       │
│       ├── Network Layer
│       │       ├── IP
│       │       ├── ICMP
│       │       └── ARP
│       │
│       └── Data Link / MAC Layer
│               ├── Ethernet
│               ├── WiFi (802.11)
│               └── PPP
│
│
└── 4. Cellular Network Generations

        ├── 1G  → FDMA

        ├── 3G → CDMA
```
