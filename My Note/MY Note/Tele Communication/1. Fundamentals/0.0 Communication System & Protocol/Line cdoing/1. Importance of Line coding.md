# Step 2 : PCM

PCM এর ভিতরে আবার 3টা ধাপ আছে:

---

## (a) Sampling

Continuous signal থেকে samples নেওয়া হয়।


$$x(t)\rightarrow x[n]$$  

---

## (b) Quantization

Amplitude কে fixed levels এ convert করা হয়।

Example:

$$256\text{ levels}$$  

---

## (c) Encoding

প্রতিটি quantized level কে binary bits এ convert করা হয়।

Example:


$$10110110$$  

এখন output হলো:

# Binary Bit Stream

---

# Important Point

PCM-এর output এখনো wire-এর জন্য proper electrical waveform না।

এটা শুধু binary data:


$$101101001011$$  

---

# Step 3 : Line Coding

এখন line coding ব্যবহার হয়।

Binary bits কে electrical pulses এ convert করা হয়।

Example:

- Unipolar
    
- Polar
    
- Bipolar AMI
    
- Manchester
    

---

# Example

PCM output:


$$10110$$  
Manchester encoding করলে:

- bit 1 → Low to High
    
- bit 0 → High to Low
    

এখন actual voltage waveform তৈরি হয়।

---

# Therefore

PCM produces:

# Digital Data

Line coding produces:

# Transmittable Electrical Signal

---

# Very Important Difference

|PCM|Line Coding|
|---|---|
|Analog → Digital|Digital → Digital waveform|
|Creates binary bits|Creates electrical pulse pattern|
|Source coding|Transmission coding|
