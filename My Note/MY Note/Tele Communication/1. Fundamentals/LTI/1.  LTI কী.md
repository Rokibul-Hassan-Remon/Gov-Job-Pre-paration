# 🔴 LTI কী?

👉  **LTI = Linear + Time-Invariant system**

---

## 🎯 2টা property

### 1. Linearity

👉  Input combine করলে output-ও combine হবে

$$x_1(t) → y_1(t), \quad x_2(t) → y_2(t)$$

👉 তাহলে:

$$x_1 + x_2 → y_1 + y_2​$$

---

### 2. Time Invariant

👉  
Input shift করলে output-ও sameভাবে shift হবে

```
Input: x(t)
Output: y(t)

Input: x(t - t0)
Output: y(t - t0)
```

---

# 🧠 এখন আসল প্রশ্ন: Role of LTI?

👉  
**Communication system-এর প্রায় সব component LTI হিসেবে model করা হয়**

---

# 🔥 1. System Analysis সহজ করে

👉 LTI হলে:

- system পুরোটা define করা যায় শুধু **impulse response** দিয়ে

$$y(t)=x(t)∗h(t)$$

👉 convolution

---

# 🔵 2. Frequency Domain Analysis possible

👉 LTI হলে:

$$Y(f)=X(f)⋅H(f)$$

👉 convolution → multiplication 😎

---

✔️ Huge benefit:

- filtering easy
- modulation analysis easy

---

# 🔴 3. Filtering (VERY IMPORTANT)

👉 Filter = LTI system

👉 Example:

- low-pass filter
- high-pass filter

👉 কাজ:

- unwanted frequency remove করা

---

# 🔵 4. Communication Channel Modeling

👉 Channel কে LTI হিসেবে ধরা হয়

```
Input signal → Channel → Output signal
```

👉 Channel = LTI system

👉 distortion, noise analysis possible

---

# 🔴 5. Sampling & Reconstruction

👉 Reconstruction filter = LTI

👉 aliasing avoid করতে:

- anti-aliasing filter use হয় (LTI)

---

# 🔥 6. Fourier Transform Relation

👉 LTI system frequency domain এ behave করে:

```
Input spectrum × System response = Output spectrum
```

👉 তাই Fourier + LTI = powerful combo

---

# 🎯 Exam Definition

👉  **An LTI system is a system that satisfies linearity and time-invariance properties, allowing signals to be analyzed using convolution in the time domain and multiplication in the frequency domain.**
