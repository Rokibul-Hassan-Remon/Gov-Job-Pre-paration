# Baud Rate Using Square Wave Signal Diagram

# CASE 1

## Binary Signaling

### (1 Signal Element = 1 Bit)

#### Input Data

```text
10110
```

#### Square Wave Signal

```text
Bits:      1       0       1       1       0

 +V      ┌─────┐         ┌─────┐ ┌─────┐
|        │     │         │     │ │     │
  0V ────┘     └─────────┘     └─┘     └────
```

#### Observation

|Bit|Signal|
|---|---|
|1|High Voltage|
|0|Low Voltage|

---

#### Total Bits

```text
5 bits
```

#### Total Signal Elements

```text
5 symbols
```

#### Result

```text
Bit Rate = Baud Rate
```

---

# CASE 2

## Multilevel Signaling

### (1 Signal Element = 2 Bits)

#### Voltage Mapping

|Voltage Level|Bits|
|---|---|
|+3V|11|
|+1V|10|
|-1V|01|
|-3V|00|

#### Input Data

```text
10110001
```

#### Grouped Bits

```text
10 | 11 | 00 | 01
```

#### Square Wave Signal

```text
 +3V          ┌────────┐
|             │        │
 +1V  ┌───────┘        └────────────────┐
      │                                 │
 -1V  │                                 └────────┐
      │                                          │
 -3V  └───────────────────────┐                  └────
|                             │
```

#### Symbol Representation

|Symbol|Voltage|
|---|---|
|10|+1V|
|11|+3V|
|00|-3V|
|01|-1V|

---

#### Total Bits

```text
8 bits
```

#### Total Signal Elements

```text
4 symbols
```

#### Formula

$$S=\frac{N}{2}$$

#### Meaning

```text
2 bits transmitted using 1 signal element
```

#### Result

- lower baud rate
    
- lower bandwidth
    
- efficient transmission
    

---

# CASE 3

## Higher Multilevel Signaling

### (1 Signal Element = 4 Bits)

#### Input Data

```text
1011001110001111
```

#### Grouped Bits

```text
1011 | 0011 | 1000 | 1111
```

#### Square Wave Signal

```text
Level 4        ┌────────┐
|              │        │
Level 3 ───────┘        └──────────┐
                                   │
|Level 2                           └──────────┐
                                              │
Level 1 ──────────────────────────────────────┘
```

#### Observation

```text
16 bits transmitted using only 4 symbols
```

#### Formula

$$S=\frac{N}{4}$$

#### Meaning

```text
4 bits transmitted using 1 signal element
```

#### Result

- very low baud rate
    
- reduced bandwidth
    
- high transmission efficiency
