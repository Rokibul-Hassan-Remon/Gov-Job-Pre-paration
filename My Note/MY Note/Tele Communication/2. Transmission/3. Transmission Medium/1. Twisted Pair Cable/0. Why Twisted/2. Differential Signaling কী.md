# ২. Differential Signaling কী?

এটাই সেই concept যেখানে receiver দুই wire-এর difference দেখে।

ধরো transmitter পাঠালো:

|Wire A|Wire B|
|---|---|
|+1V|-1V|

Receiver গণনা করল:

```
A - B
=
(+1) - (-1)
=
2V
```

এটাই data।

---

# Noise এলে কী হয়?

ধরো দুই wire-এ একই noise যুক্ত হলো:

```
Wire A = +1V + 0.3V noise
Wire B = -1V + 0.3V noise
```

Receiver:

```
(+1.3) - (-0.7)
=
2V
```

Noise বাতিল হয়ে গেল।

এটাকে বলে **Common Mode Noise Rejection**।

---

# তাহলে তোমার statement-এর কোন অংশ সত্য?

তুমি বলেছো:

> one wire is signal reference and another one is ground reference

এটা **সাধারণভাবে twisted pair-এর জন্য সঠিক নয়।**

Ethernet, USB, RS-485 ইত্যাদিতে সাধারণত:

- এক wire ground না
    
- দুই wire-ই signal বহন করে
    

উদাহরণ:

```
TX+
TX-
```

অথবা

```
D+
D-
```

অথবা

```
A
B
```

Receiver দেখে:

```
TX+ - TX-
```

---

# কখন এক wire signal আর অন্য wire ground হয়?

পুরনো telephone line বা কিছু low-speed circuit-এ এমন হতে পারে:

```
Signal
Ground
```

এটা differential signaling না।

এটাকে বলা হয় **single-ended signaling**।

উদাহরণ:

- RS-232
    
- অনেক microcontroller GPIO
    
- TTL logic
    

এখানে receiver মাপে:

```
Signal - Ground
```

---

# Twisting একা কি noise cancel করতে পারে?

অনেকেই বলে:

> "Twisting noise cancel করে"

এটা পুরোপুরি সঠিক না।

Noise rejection-এর বড় কারণ হলো:

- Differential signaling
    
- Common-mode rejection
    

আর twisting সাহায্য করে যেন:

- দুই wire একই noise receive করে
    

অর্থাৎ:

```
Twisting
    +
Differential Receiver
    =
Excellent Noise Immunity
```

এই দুইটা একসাথে কাজ করে।

# পরীক্ষার জন্য এক লাইনের উত্তর

**Twisted pair cable-এর twisting external electromagnetic noise উভয় conductor-এ প্রায় সমানভাবে induce করতে সাহায্য করে। Differential receiver দুই conductor-এর voltage difference মাপার কারণে common-mode noise বাতিল হয়ে যায়, ফলে signal integrity বৃদ্ধি পায়।**

তাই, **"receiver দুই wire-এর difference নিয়ে original signal পায়" — এটি differential signaling-এর ক্ষেত্রে সত্য। কিন্তু "এক wire signal আর অন্য wire ground" — Ethernet-এর মতো আধুনিক twisted-pair communication-এ সাধারণত সত্য নয়।**
