# Optical Fiber-এ কী হয়?

Fiber-এর দুইটা অংশ:

```text
	Core
	Cladding
```

```text
	  ┌─────────────┐
	  │    Core     │
	  └─────────────┘
	       ↓
	   Cladding
```

Core-এর refractive index একটু বেশি।

ধরো:

```text
	Core      = 1.48
	Cladding  = 1.46
```

অর্থাৎ:

```text
	Core = Light slower
	
	Cladding = Light faster
```

# কেন Core-এর index বেশি রাখা হয়?

এটাই Fiber-এর magic।

ধরো একটা light ray core-এর ভিতরে যাচ্ছে।

যখন সেটা core থেকে cladding-এ যেতে চায়:

```text
Core ---> Cladding
```

তখন refractive index change হয়।

এই কারণে light বাইরে বের না হয়ে ভেতরে bounce করে ফিরে আসে।

# 10 সেকেন্ডের intuition

Refractive Index = "আলোকে কতটা slow করা হচ্ছে"

```text
Higher Index
=
Light Slower

Lower Index
=
Light Faster
```

Fiber-এ:

```text
Core Index > Cladding Index
```

যাতে আলো বাইরে পালিয়ে না যায়।

আর:

- Step Index → index হঠাৎ কমে
    
- Graded Index → index ধীরে ধীরে কমে
    

এইটুকু বুঝলেই Optical Fiber-এর 70% concept পরিষ্কার হয়ে যাবে।
