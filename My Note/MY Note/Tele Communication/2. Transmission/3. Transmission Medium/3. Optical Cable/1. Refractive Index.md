# Refractive Index আসলে কী?

Refractive Index (n) হলো:

> **কোনো পদার্থ আলোকে কতটা slow করে দেয় তার একটি measure।**

ধরো, আলো vacuum-এ সবচেয়ে দ্রুত চলে।

এখন আলো যদি কাঁচের মধ্যে ঢোকে, তাহলে ধীরে চলবে।

যদি আরো ঘন কাঁচে ঢোকে, তাহলে আরো ধীরে চলবে।

তাই:

```text
Low Refractive Index
= Light Fast

High Refractive Index
= Light Slow
```

# একটি সহজ analogy

ধরো তুমি দৌড়াচ্ছ।

### রাস্তার উপর

```text
Speed = 20 km/h
```

### কাদার মধ্যে

```text
Speed = 5 km/h
```

কাদা তোমাকে slow করে দিচ্ছে।

ঠিক তেমনি,

```text
Material
=
Light-এর জন্য "কাদা"
```

কিছু material আলোকে কম slow করে।

কিছু material আলোকে বেশি slow করে।

এই slow করার ক্ষমতাকেই Refractive Index দিয়ে প্রকাশ করা হয়।
