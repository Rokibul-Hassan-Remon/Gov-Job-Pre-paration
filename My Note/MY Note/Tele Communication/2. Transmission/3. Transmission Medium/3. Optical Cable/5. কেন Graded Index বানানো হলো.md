# কেন Graded Index বানানো হলো?

Step Index Fiber-এ light zig-zag করে।

```text
/\/\/\/\/\/\
```

কিছু ray ছোট পথ যায়।

কিছু ray বড় পথ যায়।

ফলে:

```text
Ray A arrives first

Ray B arrives later
```

Receiver-এ signal ছড়িয়ে যায়।

---

Graded Index Fiber-এ clever একটা trick করা হয়েছে।

Outer region-এর refractive index কম।

মানে:

```text
Outer region
=
Light faster
```

Center region-এর refractive index বেশি।

মানে:

```text
Center region
=
Light slower
```

ফলে:

- যে ray বেশি দূর ঘুরে যায়, সে দ্রুত চলে।
    
- যে ray কম দূরত্ব যায়, সে ধীরে চলে।
    

শেষে তারা প্রায় একসাথে পৌঁছে।
