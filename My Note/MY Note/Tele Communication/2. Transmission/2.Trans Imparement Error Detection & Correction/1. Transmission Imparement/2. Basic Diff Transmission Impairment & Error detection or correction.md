 Transmission Impairment  & Error Detection and Correction আসলে Communication System-এর **দুইটি ভিন্ন Layer-এর সমস্যা**

---
# Transmission Impairment

এটা হলো **Problem (Cause)**

Signal Channel দিয়ে যাওয়ার সময় নষ্ট হয়।

```text
	Sender
	   |
	   v
	Transmission Medium
	   |
	   +--> Attenuation
	   +--> Distortion
	   +--> Noise
	   |
	   v
	Receiver
```

উদাহরণ:

```text
Sent     : 10110011
Received : 10100011
```

কারণ:

- Noise
    
- Distortion
    
- Attenuation
    

---

# Error Detection & Error Correction

এগুলো হলো **Solution (Countermeasure)**

Receiver চেষ্টা করে বুঝতে:

1. Error হয়েছে কি?
    
2. হলে কোথায় হয়েছে?
    
3. ঠিক করা সম্ভব কি?
    

---

## সম্পর্ক

```text
Transmission Impairment
         ↓
      Error
         ↓
Error Detection
         ↓
Error Correction
```

---

# Error Detection

Receiver শুধু বলে:

> "হ্যাঁ, Error হয়েছে।"

কিন্তু কোথায় হয়েছে জানে না।

---

উদাহরণ:

Sent:

```text
10110011
```

Received:

```text
10100011
```

Parity Check করে Receiver বুঝল:

```text
Error Found
```

কিন্তু কোন bit ভুল তা জানে না।

---

# Error Correction

Receiver বলে:

> "Error হয়েছে এবং আমি কোন bit ভুল তা জানি।"

---

উদাহরণ:

Sent:

```text
10110011
```

Received:

```text
10100011
```

Hamming Code Syndrome বের করল:

```text
Bit 4 is wrong
```

Receiver নিজেই ঠিক করল:

```text
10110011
```

---

# সহজ উপমা

ধরো তুমি বন্ধুকে SMS পাঠালে।

Original:

```text
HELLO
```

Channel-এ Noise হলো।

Receiver পেল:

```text
HELXO
```

---

### Error Detection

Receiver বলে:

```text
এই মেসেজে সমস্যা আছে।
```

কিন্তু কোথায় সমস্যা জানে না।

---

### Error Correction

Receiver বলে:

```text
৩ নম্বর অক্ষর ভুল।
সঠিক শব্দ HELLO।
```

# মূল পার্থক্য

|Transmission Impairment|Error Detection/Correction|
|---|---|
|Problem|Solution|
|Channel-এ ঘটে|Receiver-এ কাজ করে|
|Signal নষ্ট করে|নষ্ট হওয়া Signal বিশ্লেষণ করে|
|Attenuation, Distortion, Noise|Parity, Checksum, CRC, Hamming Code|
|Error তৈরি করে|Error খুঁজে বা ঠিক করে|

## Mind Map

```text
Transmission Impairment
│
├── Attenuation
├── Distortion
└── Noise
        │
        ▼
      Error
        │
        ├── Error Detection
        │      └─ "Error আছে"
        │
        └── Error Correction
               └─ "Error আছে এবং আমি ঠিক করতে পারি"
```

**সবচেয়ে গুরুত্বপূর্ণ কথা:**

Transmission Impairment নিজে Error Detection বা Correction না।

Transmission Impairment হলো **Error-এর কারণ**, আর Error Detection/Correction হলো **সেই Error মোকাবেলার পদ্ধতি**।
