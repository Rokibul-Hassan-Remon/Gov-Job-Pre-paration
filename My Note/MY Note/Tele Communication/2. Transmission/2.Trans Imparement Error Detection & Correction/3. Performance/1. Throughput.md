# 1. Throughput (Actual Data Transfer Rate)

**Throughput** হলো:

> একটি নির্দিষ্ট সময়ে বাস্তবে কত data successfully transfer হয়েছে।

### Formula


$$\text{Throughput} = \frac{\text{Total Data Transferred}}{\text{Time Taken}}  $$

### Example

10 MB data 5 second-এ transfer হলো।


$$\text{Throughput}=\frac{10\ \text{MB}}{5\ \text{s}}  =2\ \text{MB/s}$$

---

## Throughput vs Bandwidth

ধরো:

- Bandwidth = 100 Mbps
    
- বাস্তবে transfer হচ্ছে = 70 Mbps
    

তাহলে:

- Bandwidth = Maximum Capacity
    
- Throughput = Actual Achieved Rate
