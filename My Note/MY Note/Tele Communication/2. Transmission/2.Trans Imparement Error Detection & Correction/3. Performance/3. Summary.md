# Quick Summary

| Metric             | Formula                                                |
| ------------------ | ------------------------------------------------------ |
| Throughput         | $$(\frac{\text{Data}}{\text{Time}})$$                  |
| Transmission Delay | $$(\frac{\text{Packet Size}}{\text{Data Rate}})$$      |
| Propagation Delay  | $$(\frac{\text{Distance}}{\text{Propagation Speed}})$$ |
| Total Latency      | $$Processing + Queuing + Transmission + Propagation$$  |

---

# Intuition

- **Bandwidth** = রাস্তার প্রস্থ (Maximum Capacity)
    
- **Throughput** = বাস্তবে কত গাড়ি যাচ্ছে
    
- **Latency** = একটি গাড়ি Source থেকে Destination যেতে কত সময় নিচ্ছে
    

এই analogy সাধারণত Data Communication এবং Computer Network exam-এ সবচেয়ে সহজে মনে রাখা যায়।
