
# 🌳 Types of Modulation (Full Structure)

```cs title:"Modulation classification"

Modulation
│
├── 1. Analog Modulation (Continuous Wave Modulation)
│   │
│   ├── (a) Amplitude Modulation (AM)
|   |        → carrier Amplitude changes
│   │ 
|   ├── (b) Angle Modulation (AM)
|   |     │
│   │     ├──  Frequency Modulation (FM)
|   |     │      → carrier frequency changes
|   |     |   
│   │     └── Phase Modulation (PM)
│   │		     → carrier phase changes
│   |
├── 2. Digital Modulation
│   │
│   ├── (a) Amplitude Shift Keying (ASK)
│   │       → amplitude changes (0/1)
│   │
│   ├── (b) Frequency Shift Keying (FSK)
│   │       → frequency changes (0/1)
│   │
│   ├── (c) Phase Shift Keying (PSK)
│   │       → phase changes
│   │
│   │       ├── BPSK (2 phase)
│   │       ├── QPSK (4 phase)
│   │       └── DPSK
│   │
│   └── (d) QAM (Quadrature Amplitude Modulation)
│           → amplitude + phase both change
│
└── 3. Pulse Modulation
    │
    ├── (a) Analog Pulse Modulation
    │       │
    │       ├── PAM (Pulse Amplitude Modulation)
    │       ├── PWM (Pulse Width Modulation)
    │       └── PPM (Pulse Position Modulation)
    │
    └── (b) Digital Pulse Modulation
            │
            ├── PCM (Pulse Code Modulation)
            ├── DPCM
            └── Delta Modulation (DM)
```

---

# 🧠 How to remember (Smart Trick)

👉 3টা main category মনে রাখো:

1. **Analog Modulation** → continuous signal
2. **Digital Modulation** → 0 & 1
3. **Pulse Modulation** → pulses

---

# 🔥 Core idea (MOST IMPORTANT)

👉  
**Modulation মানে carrier এর কোন property change করা**

|Type|What changes|
|---|---|
|AM|Amplitude|
|FM|Frequency|
|PM|Phase|

---

# 🎯 One-line killer

👉  
**“Modulation types differ by which parameter of the carrier is varied.”**
