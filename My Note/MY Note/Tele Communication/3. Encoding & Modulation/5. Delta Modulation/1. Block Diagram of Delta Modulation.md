```txt title:"Delta Modulation"
	                                                   
						                        Feedback Path              
						                   +--------------------------+    
						                   |                          |    
						                   |       Analog Input       |    
						                   |          x(t)            |  
						                   |                          |  
						                   +------------+-------------+ 
									                    |   
						                                |                  
						                                v                  
				+----------------------+       +------------------+     
				|                      |       |                  |     
				|    Delay Unit        |-----> |    Comparator    |
				|      Previous        |       |    (+ / -)       |
				|                      |       |                  |
				+----------------------+       +------------------+
				        ^                             |
				        |                             v
				        |                  +----------------------+
				        |                  |                      |
				        |                  |  1-Bit Encoder /     |
				        |                  |     Quantizer        |
				        |                  |                      |
				        |                  +----------+-----------+
				        |                             |
				        |                             v
				        |                  +----------------------+
				        |                  |                      |
				        |                  |    Digital Output    |
				        |                  |      1 0 1 1 0       |
				        |                  |                      |
				        |                  +----------+-----------+
				        |                             |
				        |                             v
				        |                  +----------------------+
				        |                  |                      |
				        |                  |  Staircase Generator |
				        |                  |    / Integrator      |
				        |                  |                      |
				        |                  +-----------+----------+
				        |                             ^
				        |                             |
		                |-----------------------------+
		                                               
```



# Condition:
`analog signal > staircase signal -> + Delta`
				and
`analog signal < staircase signal -> - Delta`
