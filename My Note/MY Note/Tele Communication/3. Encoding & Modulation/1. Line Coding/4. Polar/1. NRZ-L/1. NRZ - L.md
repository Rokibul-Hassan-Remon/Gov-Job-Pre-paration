# 🔷 NRZ-L (Non-Return-to-Zero Level)

এখানে:

> Voltage level itself determines the bit value.

অর্থাৎ bit চিনবো voltage level দেখে।

---

### Example Convention

|Bit|Voltage|
|---|---|
|1|-V|
|0|+V|

(**Note:** কিছু বই উল্টোটাও ব্যবহার করে। Convention fixed না, principle একই।)

# 🔷 Problems of Unipolar NRZ-L

| Problem                             | Why it happens?                                                                                                                                             |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Baseline Wandering**              | Long sequence of **1s (+V)** or **0s (0V)** keeps the signal at the same level for a long time, causing the receiver's reference level (baseline) to drift. |
| **No Self-Synchronization**         | Long runs of **1s** or **0s** create no transitions, so the receiver cannot recover or maintain its clock timing.                                           |
| **DC Component**                    | The average voltage is not zero because the signal uses **+V for 1** and **0V for 0**, resulting in a non-zero DC level.                                    |
| **Greater Normalized Signal Power** | Bit **1** is transmitted using **+V for the entire bit duration**, which increases the average transmitted power.                                           |
