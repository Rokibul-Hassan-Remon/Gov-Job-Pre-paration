Polar RZ (Return-to-Zero) বুঝতে হলে আগে নামটাই ভাঙি:

	
- **Polar** → +V এবং -V দুইটাই ব্যবহার করে
    
- **RZ (Return to Zero)** → প্রতিটি bit-এর মাঝখানে signal **0V-তে ফিরে আসে**
	

---

# 🔷 Polar RZ Rule

একটি common convention:

| Bit | Signal  |
| --- | ------- |
| 1   | +V → 0V |
| 0   | -V → 0V |

অর্থাৎ bit time-এর প্রথম অর্ধেক voltage থাকে, তারপর দ্বিতীয় অর্ধেকে 0V-তে ফিরে যায়।

# 🧠 Intuition

**NRZ:**

```text
	1 = +V ─────────
```

পুরো bit duration জুড়ে level একই।

---

**RZ:**

```text
	1 = +V ────┐
	           └── 0V
```

bit শেষ হওয়ার আগেই zero-তে ফিরে আসে।

সেজন্য নাম:

```text
	RZ = Return To Zero
```

---

# 🔷 Why was RZ introduced?

NRZ-এর সমস্যা ছিল:

- Long runs of same bits
    
- Synchronization problem
    

RZ বলল:

> "আমি প্রতি bit-এ zero-তে ফিরে যাব, যাতে transition বেশি হয়।"

ফলে clock recovery সহজ হয়।

---

# 🔷 Advantages of Polar RZ

|Advantage|Why it happens?|
|---|---|
|Better Synchronization|প্রতি bit-এ zero-তে return করায় বেশি transitions পাওয়া যায়|
|Less Baseline Wandering|Signal দীর্ঘ সময় একই level-এ থাকে না|
|Easier Clock Recovery|Receiver bit boundary সহজে বুঝতে পারে|

---

# 🔷 Problems of Polar RZ

|Problem|Why it happens?|
|---|---|
|Higher Bandwidth Requirement|প্রতি bit-এ extra transition তৈরি হয়|
|Higher Signal Rate|More transitions than NRZ|
|More Complex|Three voltage levels (+V, 0V, -V) handle করতে হয়|

---

# 🔥 Compare NRZ vs RZ

|Feature|Polar NRZ|Polar RZ|
|---|---|---|
|Voltage Levels|+V, -V|+V, 0V, -V|
|Return to Zero|❌ No|✔ Yes|
|Synchronization|Poor|Better|
|Bandwidth|Lower|Higher|
|Number of Transitions|Fewer|More|

---

# 🧠 Memory Trick

### NRZ

```text
	NRZ = Never Returns to Zero
```

(আসলে bit-এর মাঝখানে return করে না)

---

### RZ

```text
	RZ = Returns to Zero
```

প্রতি bit-এ:

```text
	+V → 0V
	
	   or
	   
	-V → 0V
```

---

# ⚡ Exam-Oriented Summary

|Problem|Why it happens?|
|---|---|
|Higher Bandwidth Requirement|Signal returns to zero in every bit, creating more transitions|
|Higher Signal Rate|More transitions per bit than NRZ|
|More Complex Implementation|Uses three signal levels (+V, 0V, -V)|

### One-line definition

> **Polar RZ is a line coding scheme that uses positive and negative voltages and forces the signal to return to zero during each bit interval, improving synchronization at the cost of higher bandwidth.**
