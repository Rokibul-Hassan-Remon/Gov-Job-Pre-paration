**Multi-Level Line Coding** হলো এমন line coding scheme যেখানে **দুইটির বেশি signal level** ব্যবহার করা হয়।

আগে যা দেখেছ:

- Unipolar → 2 level (`0`, `+V`)
    
- Polar → 2 level (`+V`, `-V`)
    
- Bipolar → 3 level (`+V`, `0`, `-V`)
    

এখন Multi-Level coding-এ 3, 4, 5 বা তারও বেশি level ব্যবহার করা হতে পারে।

---

# কেন Multi-Level Coding দরকার হলো?

ধরো Binary system:

```text
	0
	1
```

প্রতি signal element (baud) = 1 bit

---

কিন্তু যদি 4টি voltage level ব্যবহার করি:

```text
	-3V
	-1V
	+1V
	+3V
```

তাহলে:

```text
	00
	01
	10
	11
```

একটি signal element দিয়েই 2 bit পাঠানো সম্ভব।

---

# মূল ধারণা

যদি `signal level `সংখ্যা = **L** হয়

তাহলে প্রতি signal element-এ পাঠানো যাবে:

$$Bits \ per \ signal\ element = \log_2(L)$$

উদাহরণ:

|Level (L)|Bits/Symbol|
|---|---|
|2|1 bit|
|4|2 bit|
|8|3 bit|
|16|4 bit|

---

# Intuition

ধরো 8 Mbps data পাঠাতে হবে।

### Binary NRZ

```text
1 symbol = 1 bit
```

তাহলে:

```text
8 Mbps = 8 Mbaud
```

---

### 4-Level Coding

```text
1 symbol = 2 bit
```

তাহলে:

```text
8 Mbps = 4 Mbaud
```

Signal rate অর্ধেক হয়ে গেল।

ফলে কম bandwidth লাগে।

---

# Advantage

## 1. Bandwidth কম লাগে

কারণ:

```text
এক symbol-এ বেশি bit
```

---

## 2. High-Speed Transmission

একই bandwidth-এ বেশি data পাঠানো যায়।

---

# Disadvantage

## Receiver Complexity

Receiver-কে level distinguish করতে হয়।

2 level হলে:

```text
+V না -V?
```

সহজ।

---

4 level হলে:

```text
-3V?
-1V?
+1V?
+3V?
```

কঠিন।

---

## Noise Sensitivity

ধরো:

```text
+1V level
```

Noise এসে:

```text
+1.7V
```

করে দিল।

Receiver ভুল করে `+3V` ধরে ফেলতে পারে।

Level যত বেশি:

```text
Gap তত ছোট
```

Gap যত ছোট:

```text
Noise effect তত বেশি
```

---

# Common Multi-Level Schemes

## 1. 2B1Q

নাম:

```text
2 Binary
1 Quaternary
```

অর্থাৎ:

```text
2 bit → 1 symbol
```

4 level ব্যবহার করে।

Mapping:

|Bits|Signal|
|---|---|
|00|-3V|
|01|-1V|
|10|+1V|
|11|+3V|

---

## 2. 8B6T

নাম:

```text
8 Binary
6 Ternary
```

8 bit data → 6 ternary symbols।

---

## 3. MLT-3 (Multi-Level Transmit-3)

Level:

```text
+V
0
-V
```

Sequence:

```text
0 → no change

1 → next level
```

Level cycle:

```text
0
→ +V
→ 0
→ -V
→ 0
→ +V
```

এটি 100 Mbps Ethernet-এ ব্যবহৃত হয়েছে।

# পরীক্ষার জন্য One-Line Definition

**Multi-Level Line Coding** হলো এমন line coding technique যেখানে দুইটির বেশি signal level ব্যবহার করে এক signal element-এ একাধিক bit encode করা হয়, ফলে signal rate ও bandwidth requirement কমে যায়।
