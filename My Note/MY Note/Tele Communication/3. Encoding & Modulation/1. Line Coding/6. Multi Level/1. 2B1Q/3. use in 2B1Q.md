**2B1Q মূলত টেলিফোন কোম্পানিগুলো copper twisted-pair cable-এর উপর high-speed digital data পাঠানোর জন্য ব্যবহার করত।**

সবচেয়ে গুরুত্বপূর্ণ ব্যবহারগুলো ছিল:

### 1. ISDN (Integrated Services Digital Network)

বিশেষ করে ISDN-এর **U Interface**-এ 2B1Q ব্যাপকভাবে ব্যবহৃত হয়েছে। North America, Italy এবং Switzerland-এ এটি standard line coding হিসেবে ব্যবহৃত হতো।

---

### 2. HDSL

HDSL-এর প্রথম দিকের implementation-গুলোতে 2B1Q ব্যবহৃত হতো যাতে বিদ্যমান copper telephone line ব্যবহার করেই T1/E1 digital service চালানো যায়।

---

# কেন 2B1Q ব্যবহার করা হতো?

AMI-তে:

```
1 bit → 1 symbol
```

কিন্তু 2B1Q-তে:

```
2 bit → 1 symbol
```

ফলে:

```
Bit Rate = 160 kbps
```

হলে

```
Symbol Rate = 80 kbaud
```

হয়ে যায়। অর্থাৎ baud rate অর্ধেক হয়ে যায় এবং কম bandwidth লাগে।

---

# আজকাল কি 2B1Q ব্যবহার হয়?

আধুনিক Ethernet, Fibre Optic, 4G/5G ইত্যাদিতে সাধারণত 2B1Q ব্যবহার হয় না।

তবে 2B1Q খুবই গুরুত্বপূর্ণ কারণ:

- এটি Multi-Level Line Coding-এর একটি classic example।
- PAM-4 (যা modern high-speed communication-এ ব্যবহৃত হয়) বুঝতে 2B1Q একটি foundation হিসেবে কাজ করে।
- DSL এবং ISDN প্রযুক্তির ইতিহাসে এটি একটি গুরুত্বপূর্ণ milestone।
