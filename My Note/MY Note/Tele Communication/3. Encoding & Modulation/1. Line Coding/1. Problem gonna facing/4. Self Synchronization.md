# Self Synchronization in Line Coding

## Real Problem

Receiver শুধু voltage waveform দেখে।

সে জানে না:

```text
কখন একটি bit শুরু
আর কখন শেষ
```

---

# Problem Example

## Unipolar NRZ

### Input

```text
111111111
```

### Waveform

```text
 +V ───────────────────────
```

---

# Problem

## No Transition

### Receiver Confused

Signal অনেকক্ষণ change না হলে receiver বুঝতে পারে না:

```text
bit boundary কোথায়
```

ফলে:

- clock drift
    
- wrong sampling
    
- synchronization loss
    

---

# Solution

## Self Synchronization

### Main Idea

Signal-এর ভিতরেই timing information রাখা হয়।

---

# How?

## Using Transitions

### Transition Means

```text
High ↔ Low change
```

প্রতিটি transition receiver-কে timing clue দেয়।

---

# Example : Manchester Encoding

## Rule

|Bit|Signal|
|---|---|
|1|Low → High|
|0|High → Low|

---

# Waveform

## Input

```text
1010
```

### Signal

```text
 _|‾    ‾|_    _|‾    ‾|_
```

---

# Important Observation

## Every Bit Has Transition

### Therefore

Receiver সহজে বুঝতে পারে:

```text
new bit এসেছে
```

---

# Final Core Understanding

## Self Synchronization Means

Signal waveform-এর ভিতরেই:

```text
clock/timing information embedded থাকে
```

through:

```text
regular voltage transitions
```

so receiver can correctly detect bit boundaries.
