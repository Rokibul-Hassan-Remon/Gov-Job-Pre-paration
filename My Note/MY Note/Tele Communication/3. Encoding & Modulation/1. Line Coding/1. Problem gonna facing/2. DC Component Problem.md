# DC Component Problem

## Which End This Problem Occurs?

### Transmission Channel

#### Reason

Communication channel-এর ভিতরে থাকে:

- capacitor
    
- transformer
    
- coupling circuit
    

এগুলো DC signal pass করতে পারে না।

## Physical Meaning

### Constant Voltage

#### Important Observation

এখানে signal অনেকক্ষণ একই voltage-এ আছে।

মানে:

```text
frequency ≈ 0 Hz
```

এটি DC-like behavior তৈরি করে।
