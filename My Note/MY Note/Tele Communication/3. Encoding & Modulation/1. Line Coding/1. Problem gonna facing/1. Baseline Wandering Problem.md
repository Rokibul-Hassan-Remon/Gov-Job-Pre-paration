# Baseline Wandering Problem

## Which End This Problem Occurs?

এই problem mainly দেখা যায়:

- receiver side-এ
    
- transmission channel-এ
    
- capacitor coupled system-এ
    
- transformer coupled system-এ
    

especially when:

```text
long sequence of same bits occurs
```


#### Reason

Receiver incoming signal দেখে একটি threshold/reference voltage ব্যবহার করে decide করে:

```text
Voltage > Threshold  →  Bit 1

Voltage < Threshold  →  Bit 0
```

Example threshold:

```text
0.9V to 4V      →  1
Less than 0.9V →  0
```

---

## How This Problem Occurs?

### Long Sequence of Same Bits

#### Example

Input Data:

```text
111111111111
```

## What Receiver Thinks?

### Receiver Moving Average

#### Physical Effect

Receiver অনেকক্ষণ same high voltage দেখে ভাবতে শুরু করে:

```text
"Maybe this is the new normal baseline"
```

তাই receiver threshold ধীরে ধীরে উপরের দিকে shift হয়।

---

## Graphical Baseline Shift

### Threshold Drift

#### Baseline Wandering

```text
Original Threshold

		0.9V ───────────────────────────


After Long High Voltage
		1.8V ─────────────────────────── ↗ Shifted Threshold
```

---

## Final Problem

### Wrong Detection

#### Receiver Confusion

এখন যদি weak signal আসে:

```text
Actual Bit = 1
```

কিন্তু shifted threshold-এর কারণে receiver detect করতে পারে:

```text
Detected Bit = 0
```
