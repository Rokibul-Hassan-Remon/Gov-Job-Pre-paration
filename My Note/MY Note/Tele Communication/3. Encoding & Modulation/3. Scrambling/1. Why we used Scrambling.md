
# 1. Scrambling

### Definition

Long sequence of 0s বা 1s আসলে সেগুলোকে special patterns দিয়ে replace করা হয় যাতে synchronization বজায় থাকে।

### How it works?

```text
Data
 ↓
Scrambler
 ↓
Line Coding
 ↓
Transmission
```

### Advantages

✅ Self-Synchronization বজায় রাখে

✅ Long-distance transmission-এর জন্য উপযোগী

✅ Extra bits যোগ করে না

### Disadvantages

❌ Error Detection দেয় না

### Examples

- B8ZS
    
- HDB3
    

---

## Which Line Coding uses Scrambling?

Scrambling মূলত **AMI (Alternate Mark Inversion)** এর সাথে ব্যবহৃত হয়।

কারণ AMI-এর প্রধান সমস্যা:

```text
00000000
```

↓

```text
0V 0V 0V 0V 0V 0V 0V 0V
```

কোন transition নেই।
