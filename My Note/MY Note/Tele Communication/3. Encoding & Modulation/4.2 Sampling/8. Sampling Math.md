# PCM / Sampling Note

# Nyquist Rate and Discrete-Time Signal

---

# Question

A continuous signal is given below:


$$x(t)=8\cos(2\pi100t)$$  
Find:

1. Nyquist Rate
    
2. Discrete-time signal (x[n]) when:
    


$$f_s=400\text{ Hz}$$  

---

# Step 1 : Identify the Signal Frequency

Standard cosine signal form:

$$x(t)=A\cos(2\pi ft+\phi)$$

Where:

- (A) = Amplitude
    
- (f) = Frequency
    
- (\phi) = Phase
    

Now compare:

[  
$$x(t)=8\cos(2\pi100t)$$  
]

Therefore,

Amplitude:

[  
$$A=8$$  
]

Frequency:

[  
$$f=100\text{ Hz}$$  
]

---

# Step 2 : Find Nyquist Rate

Nyquist rate formula:

$$f_s=2f_{max}$$

Maximum frequency of the signal:

[  
$$f_{max}=100\text{ Hz}$$  
]

Therefore,

[  
$$f_s=2\times100$$  
]

[  
$$f_s=200\text{ Hz}$$  
]

# Answer

[  
$$\boxed{\text{Nyquist Rate}=200\text{ Hz}}$$  
]

---

# Step 3 : Find Sampling Period

Given:

[  
$$f_s=400\text{ Hz}$$  
]

Sampling period formula:

[  
$$T_s=\frac1{f_s}$$  
]

So,

[  
$$T_s=\frac1{400}$$  
]

[  
$$T_s=0.0025\text{ sec}$$  
]

---

# Step 4 : Convert Continuous Signal to Discrete Signal

In sampling:

[  
$$t=nT_s$$  
]

Discrete signal is obtained by:

[  
$$x[n]=x(nT_s)$$  
]

Substitute into the original signal:

[  
$$x[n]=8\cos(2\pi100(nT_s))$$  
]

Now substitute:

[  
$$T_s=\frac1{400}$$  
]

Then,

[  
$$x[n]=8\cos\left(2\pi100\left(\frac n{400}\right)\right)$$  
]

Simplify:

[  
$$x[n]=8\cos\left(\frac{200\pi n}{400}\right)$$  
]

[  
$$x[n]=8\cos\left(\frac{\pi n}{2}\right)$$  
]

# Final Discrete-Time Signal

$$x[n]=8\cos\left(\frac{\pi n}{2}\right)$$

---

# Step 5 : Calculate Sample Values

---

For (n=0)

[  
$$x[0]=8\cos(0)=8$$  
]

---

For (n=1)

[  
$$x[1]=8\cos\left(\frac{\pi}{2}\right)=0$$  
]

---

For (n=2)

[  
$$x[2]=8\cos(\pi)=-8$$  
]

---

For (n=3)

[  
$$x[3]=8\cos\left(\frac{3\pi}{2}\right)=0$$  
]

---

For (n=4)

[  
$$x[4]=8\cos(2\pi)=8$$  
]

---

# Sample Sequence

[  
$$8,;0,;-8,;0,;8,\dots$$  
]

---

# Important Physical Interpretation

Original signal frequency:

[  
$$f=100\text{ Hz}$$  
]

Sampling frequency:

[  
$$f_s=400\text{ Hz}$$  
]

Therefore,

[  
$$\frac{f_s}{f}=4$$  
]

Meaning:

- 4 samples are taken per cycle.
    

These samples occur at:

1. Positive Peak
    
2. Zero Crossing
    
3. Negative Peak
    
4. Zero Crossing
    

That is why the sequence becomes:

[  
$$8,;0,;-8,;0,;8,\dots$$  
]

---

# Final Summary

| Parameter          | Value                                        |
| ------------------ | -------------------------------------------- |
| Signal Frequency   | $$(100\text{ Hz})$$                          |
| Nyquist Rate       | $$(200\text{ Hz})$$                          |
| Sampling Frequency | $$(400\text{ Hz})$$                          |
| Sampling Period    | $$(0.0025\text{ sec})$$                      |
| Discrete Signal    | $$(x[n]=8\cos\left(\frac{\pi n}{2}\right))$$ |
