# Quantization Error Formula

Quantization error:

$$e_q = x - x_q$$

Where:

- (e_q) = quantization error
    
- (x) = original analog sample value
    
- (x_q) = quantized value
    

---

# Maximum Quantization Error
### Quantization Step Size

Uniform quantizer:

$$\Delta = \frac{V_{max}-V_{min}}{L}$$
### Quantization Error

$$|e_q|_{max}=\frac{\Delta}{2}$$

Where:

- (Δ ) = quantization step size
    
- (L) = number of quantization levels
    

---

# Number of Levels

If (n)-bit PCM:

$$L=2^n$$
# RMS Quantization Error

$$$e_{rms}=\frac{\Delta}{\sqrt{12}}​$$

# Therefore Noise Power

Substitute RMS value:

$$P_q=\left(\frac{\Delta}{\sqrt{12}}\right)^2$$

So,

$$P_q=\frac{\Delta^2}{12}​$$
