# Bit Rate

Bit rate means:

> Number of bits transmitted per second.

Unit:

$$\text{bps} = \text{bits per second}  $$

---

# Bit Rate Formula

$$R_b = n f_s$$

Where:

- (Rb) = Bit rate
    
- (n) = Number of bits per sample
    
- (fs) = Sampling rate
    

---

# PCM Bit Rate

**For PCM system:**

$$R_b = n \times 2f_{max}$$

because:

$$f_s = 2f_{max}  $$

---

# Bandwidth

Bandwidth means:

> Range of frequencies occupied by a signal or channel.

---

# Communication Bandwidth

$$BW = f_H - f_L$$

Where:

- (f_H) = Highest frequency
    
- (f_L) = Lowest frequency
    

---

# PCM Transmission Bandwidth

Approximate PCM bandwidth:

$$BW_{PCM}=\frac{R_b}{2}$$

Sometimes written as:

 
$$BW=nf_{max}  $$


depending on line coding method.

---

# SNR Formula for Sinusoidal Signal in PCM

For a full-scale sinusoidal input signal, the Signal-to-Quantization Noise Ratio (SQNR or SNR) is:

$$SNR_{dB}=6.02n+1.76$$

Where:

- (n) = number of bits per sample
    
- (SNR_{dB}) = Signal-to-noise ratio in decibel
    

---

# Linear Form

$$SNR=\frac{P_s}{P_q}$$

Where:

- (P_s) = signal power
    
- (P_q) = quantization noise power
    

---

# Quantization Noise Power

$$P_q=\frac{\Delta^2}{12}$$

---

# Sinusoidal Signal Power

For sinusoidal signal:

$$P_s=\frac{A^2}{2}$$

Where:

- (A) = peak amplitude
    

---

# Therefore SNR

$$SNR=\frac{A^2/2}{\Delta^2/12}$$

Simplifying:

$$SNR=\frac{6A^2}{\Delta^2}$$

---

# SNR in dB

$$SNR_{dB}=10\log_{10}\left(\frac{P_s}{P_q}\right)$$

---

# Important PCM Relations Table

| ------------------------ | --------------------------- |

| PCM Sinusoidal SNR       | $$(6.02n+1.76\ \text{dB})$$ |
