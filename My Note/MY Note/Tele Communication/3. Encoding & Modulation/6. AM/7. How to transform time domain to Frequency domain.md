# 1. Time Domain Signal

ধরি carrier:

$$c(t)=A_c\cos(2\pi f_c t)$$

---

# 2. Time Domain এ এটা দেখতে কেমন?

```text
Amplitude
   ^
Ac |      /\      /\      /\      /\
   |     /  \    /  \    /  \    /  \
 0 +----/----\--/----\--/----\--/----\----> t
   |   /      \/      \/      \/      \
-Ac|
```

👉 এখানে:

- x-axis = time
    
- y-axis = amplitude
    
- wave oscillate করছে
    

---

# 3. Fourier Transform কী করে?

Fourier Transform জিজ্ঞেস করে:

> “এই signal-এর ভিতরে কোন frequency আছে?”

Pure cosine এ only one frequency আছে:

 
$$f_c  $$


---

# 4. Frequency Domain Spectrum

Fourier Transform করার পরে:
1. General rule
$$c(t) = A_c\cos(2\pi f_c t) \leftrightarrow \frac{A_c}{2}[\delta(f-f_c)+\delta(f+f_c)] = c(f)$$

```text

 ^(Amplitude)
 |
 |                    ↑
 |                    |
-+--------------------|-------------------> f
 |                   +fc
```

কিন্তু mathematically full spectrum actually:

```text
Amplitude

|                            ^c(f)
|                            |                           
|           (Ac/2)↑          |          ↑(Ac/2)
|                |           |          |
------+----------|----------------------|--------> f
                -fc                   +fc
```

---

# 5. কেন দুইটা spike?

কারণ:

$$\cos(2\pi f_c t)=\frac{1}{2}(e^{j2\pi f_c t}+e^{-j2\pi f_c t})$$

মানে cosine আসলে:
	
- positive rotating frequency
    
- negative rotating frequency
    

দুইটার combination।

# 7. AM হলে spectrum কেমন হয়?

- carrier = (f_c)
    
- message = (f_m)

```text
Amplitude

                 LSB    Carrier      USB
```
