# Block Coding & Scrambling (Exam/Viva Note)

## Why do we need them?

অনেক Line Coding scheme-এ **long sequence of 0s বা 1s** আসলে receiver clock synchronization হারাতে পারে।

এ সমস্যা সমাধানের জন্য ব্যবহার করা হয়:

1. **Block Coding**
    
2. **Scrambling**
    

---

### 1. Block Coding

#### Definition

`m` data bits কে `n` coded bits-এ (`n > m`) রূপান্তর করা হয় যাতে পর্যাপ্ত transition থাকে এবং কিছু error detection সম্ভব হয়।

#### How it works?

```text
		Data
		 ↓
	Block Coding
		 ↓
	Line Coding
		 ↓
	Transmission
```

#### Advantages

✅ Self-Synchronization improve করে

✅ Limited Error Detection দেয়

✅ Long runs of 0s/1s কমায়

#### Disadvantages

❌ Extra bits যোগ হয় (Overhead)

#### Examples

- **4B/5B**
    
- **8B/10B**
    

---

#### Which Line Coding uses Block Coding?

|Block Coding|Usually Used With|
|---|---|
|4B/5B|NRZ-I|
|8B/10B|NRZ-based systems, High-Speed Serial Links|

#### Example

```text
	Data
	 ↓
   4B/5B
	 ↓
   NRZ-I
	 ↓
   Signal
```

# Quick Comparison

| Feature          | Block Coding                      | Scrambling      |
| ---------------- | --------------------------------- | --------------- |
| Extra Bits Added | ✅ Yes                             | ❌ No            |
| Synchronization  | ✅ Yes                             | ✅ Yes           |
| Error Detection  | ✅ Limited                         | ❌ No            |
| Main Purpose     | Synchronization + Error Detection | Synchronization |
| Used With        | NRZ-I                             | AMI             |
| Examples         | 4B/5B, 8B/10B                     | B8ZS, HDB3      |

---

# Most Important Exam Point

| **Line Coding Problem**                   | **Solution**            |
| ----------------------------------------- | ----------------------- |
| NRZ-I has synchronization problem         | 4B/5B Block Coding      |
| AMI has long-zero synchronization problem | B8ZS or HDB3 Scrambling |
|                                           |                         |

#### Easy Memory Trick

```text 

		NRZ-I
		  ↓
	Block Coding
		  ↓
		4B/5B
	  
```


```text 

	 AMI
	  ↓
  Scrambling
	  ↓
  B8ZS / HDB3
  
```

**এক লাইনে:**

> **Block Coding adds extra bits before NRZ-type line coding to improve synchronization and provide limited error detection, whereas Scrambling replaces problematic patterns in AMI without adding extra bits to maintain synchronization.**
