> **Primary Key হলো একটি SQL Constraint (enforcement mechanism) যা Relational Model-এর Entity Integrity Constraint enforce করে। এটি table-এর প্রতিটি row-কে uniquely identify করে, duplicate value allow করে না এবং NULL value allow করে না। সাধারণত এটি Candidate Key থেকে নির্বাচিত হয় এবং table-এর প্রধান identity হিসেবে কাজ করে।**

আরও ছোট viva answer:

> **Primary Key is a unique and non-null identifier for each row in a table. In relational theory, it is used to enforce the Entity Integrity Constraint, and in SQL it is implemented through the PRIMARY KEY constraint.**
