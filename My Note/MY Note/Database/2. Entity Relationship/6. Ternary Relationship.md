
## 3. Ternary Relationship (Degree = 3)

তিনটি Entity একসাথে relationship-এ অংশগ্রহণ করে।

```text
Supplier
    \
     \
      Supplies
     /    |    \
    /     |     \
Part   Project
```

উদাহরণ:

- Supplier supplies Part to Project
    

এখানে:

- Supplier
    
- Part
    
- Project
    

মোট ৩টি Entity আছে, তাই **Ternary Relationship**।

### কেন Binary দিয়ে হবে না?

কারণ:

```text
Supplier → Part
Part → Project
```

লিখলে বোঝা যায় না **কোন Supplier কোন Project-এ কোন Part সরবরাহ করেছে**।

তিনটি Entity-র তথ্য একসাথে দরকার, তাই Ternary।
